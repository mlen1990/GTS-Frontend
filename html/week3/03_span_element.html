Span Elements

This is a paragraph tag. You might use a span to make red text.
You can also underline certain parts of the text.
There are other things you can do using the text-decoration property, such as a strikethrough.
Another thing you can do is add some whitespace between words.
Highlighting some words can be done using the background property.
I don't know how often this is used but you can put multiple formatting options together if they use different CSS properties.
There are a lot of possibilties.
